# Projeto Agenda #

> Linguagem utilizada: **Python**
> Framework: **Django***

### Iniciando o projeto ###

1) Após clonar o repositório, criar o ambiente virtual:

    python -m venv venv


2) Iniciar o ambiente virtual:

    venv/Scripts/activate


3) Instalar as dependências necessárias:

    pip install django


4) Rodar o projeto:

    python manage.py runserver
